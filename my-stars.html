My potentially useful Github stars

I have some things starred on Github. These are ones that might someday be useful, mostly for my own reference.

JavaScript

  • Temporal - better datetimes for JavaScript, coming natively to browsers.
  • Effect-TS - Pydantic for TypeScript, maybe better than io-ts? See also zod, trpc.
  • Proposal to have erasable type annotations supported natively in the browser.
  • daisyui - nice looking Tailwind components.
  • engine262 - JS interpreter in JS.
  • morphdom - morph one DOM element into another, no VDOM.
  • flamebearer - flame graph profiling.
  • shoelace - nice looking frontend component library.

Python

  • ruff - nouveau linting a la flake8.
  • snakeviz - flame graph profiling.
  • pyinstrument - runtime profiling
  • scalene - nouveau profiling hotness
  • greenery - determine the minimal possible regex/regex equality.
  • MonkeyType - generate type annotations by running tests.
  • spectree - FastAPI for Flask.
  • dtparse - example PyO3 usage (Rust Python wrapper).
  • pex - bundle Python programs.
  • Strictly typed Pandas.
  • Meta Pydantic.
  • Convex Optimization.
  • mashumaro - fast dataclass serialization - probably not relevant since Pydantic speedups.

Developer tools

  • swimm.io - funky developer docs.
  • difftastic - nice diff tool.
  • remark - easy Markdown powerpoints.
  • marimo - alternative to Jupyter notebooks.
  • superset - dashboard builder builder and in-browser SQL IDE.

SQL/Postgres

  • Graph Queries are coming to ANSI SQL.
  • pgcli alternative to psql.
  • Postgres++ for Kubernetes - probably a good starting point for backups/connection pooling etc.
  • Postgres + Elasticsearch.
  • Rust SQL parser, used by gluesql.
  • realtime - listen to Postgres changes over Websockets.
  • Magic F# SQL typechecking.

Incremental View Maintenance

  • pg_ivm - IVM extension to Postgres, this might be the route it takes to end up native.
  • dbsp implementation, see theory.
  • noria - materialize.io for MySQL applications.

Other

  • AudioSep - split audio files into stems.
  • catala - law as code.
